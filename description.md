# ゲームの概要

## 根本的な概要
- 2Dアクションゲーム
- 物理による表現にこだわりがある
- スマホとPCの両方で操作できる
- ブラウザで動くがオンライン要素はない
- キャラクタはLive2Dのようにボーンベースでなめらかに動く

## キャラクター設定
- キャラクターは次の2キャラクター
  - 主人公：親指トムのようにとても小さく、後述するヤギの鼻の穴の直径の半分より少し小さい。プレイヤーが操作するキャラクターで小型の人型。
  - 巨大ヤギ：referenceAssets/imagesにあるようなヤギ。絶対的な大きさは普通のヤギだが、小さな主人公から見ればゴジラのような大きさに見える。未知の小さな生き物を見つけるとまず匂いを嗅ごうとするが、たいがいはその吸引力で誤って鼻の中に吸い込んでしまう。主人公のような小さなものが鼻の中の壁に当たったり、中の鼻毛を抜かれたりすると鼻がムズムズして、ムズムズの度合いが一定量を超えるとくしゃみをしてしまう。

## プロローグ
- 舞台は牧場のヤギ舎。主人公は干し草の上でお気に入りのぬいぐるみを抱いてゆっくりしていて、ヤギは辺りを歩いている。
- 巨大ヤギが主人公を見つけて不思議がり、主人公のほうへ歩いていく。
- (カメラを主人公目線に切り替える)ヤギは興味を持って顔を主人公の真ん前まで近づける(この時主人公の前には巨大なヤギの鼻のドアップでヤギは主人公に対して正面を向いている)。
- 主人公がびっくりしたのもつかの間、(ここでエフェクトとしてカメラを小刻みに動かし始めながら主人公の大きさが変わらないようにカメラをズームして追いかける)ヤギが鼻からの巨大な吸い込みを始め(このときヤギの鼻の動きはボーンベースでなめらかにうごかすようにする)、主人公たちはあっけなく巨大なヤギの鼻の穴の中に吸い込まれてしまう(最終的に画面全体が真っ暗にする)。
- ヤギの鼻の中で目覚めた主人公だが、大事なぬいぐるみがなくなっているのに気づく。どうやらヤギの鼻の奥に詰まってしまったらしい。ヤギの鼻の中の壁をあちこち触るとくしゃみをしてしまい、外に弾き飛ばされてしまう。ヤギの鼻の中を刺激しないよう鼻の奥まですすまなければならない。主人公の小さな冒険が始まる。

## ゲームの詳細な設定
- 舞台はプロローグのようにヤギの鼻の中。goatRight.pngの顔部分を拡大した部分を背景のように使うが、ここもボーンベースで、ヤギの顔や口がなめらかに動く。
- 舞台の地形はいりくんだ洞窟のような形に鼻毛がまだらに生えている。具体的には、まず洞窟部分は円形のチューブが鼻の穴の入り口から鼻の奥(ちょうどヤギの耳の位置あたり)まで伸びていて、その半径は平均が主人公の大きさよりわずかに大きい程度で場所によって少し大きさが大きいまたは小さい場所があり、さらに上方向や下方向に少しだけグネグネしている。そして鼻毛はある程度等間隔に地面または天井から、主人公と同程度の長さで生えている。鼻の中は少し弾力があるような物理挙動をし、鼻毛もなびくようなばねのような物理挙動をする。ヤギは呼吸などに応じて顔の角度を上下に変えるが、鼻の洞窟もそれに追従するように角度を変える。ゲームスタート時は、洞窟が水平になるように顔の角度、洞窟の角度を調整する。
- ヤギにはくすぐりメーターが存在し、これがたまると大きなくしゃみをする。常に少しずつメーターがなくなっていくが、主人公が地面を歩いたり、地面や天井にぶつかるごとにこのメーターがたまる。また後述する鼻毛が抜けた場合には一気にくしゃみメーターが満タンまでたまる。メーターがたまるとヤギは短い連続した呼吸を3回してそのあと大きなくしゃみを行うが、この間でヤギの顔の角度はまず短い呼吸で都度上方向に細かく角度をつけて動かし、最後のくしゃみで急速に顔の角度を元の水平に戻す。くしゃみをした瞬間にメーターを0にして、以降はメーターの運用を再開する。くしゃみの音についてはreferenceAssets/soundsにある音を鳴らしてください。三回の短い連続した呼吸はsneeze1.wav,sneeze2.wav,sneeze3.wav、一回の大きなくしゃみはsneeze_last.wavにしてください。
- 主人公に対しては左右移動、ジャンプ、うつ伏せになる、鼻毛に掴まるボタン(またはそれに最適なUI)が設けられている。主人公はうつ伏せ状態と直立状態の二つがあり、スタート時は直立状態である。直立状態はy軸平行に体を保っていてこの状態で左右移動を、うつ伏せ時より早く動けてよりヤギの鼻を刺激しない。うつ伏せ状態だと地面に常にそうような恰好になり、左右移動が遅くなり動くとヤギの鼻をより刺激する。ジャンプはどちらの状態でも性能差なしで、天井にぎりぎりぶつからない程度のジャンプ力で行うことができ、うつ伏せ状態であっても直立状態に変更する。鼻毛が近くにある時は鼻毛に掴まるボタンで鼻毛の先端に追従するように掴まることができ、この間は動くことができない、もう一度同じボタンを押すか、後述する鼻毛が切れるようなことがない限り離さない。ヤギによる巨大なくしゃみが起こると、主人公は鼻の穴の入り口に向かって大きな速度が加算される。
- ヤギの鼻の中にある鼻毛は主人公がつかむことができるが、それぞれ耐久メータが存在する。主人公が掴まっている間はたとえ主人公が動いていなくともメーターが少しずつなくなっていき、主人公の速度に比例してメーターがさらに減る。メーターがなくなるとその鼻毛が抜けてしまい、主人公のつかみ状態が解除されてしまうとともにヤギのくすぐりメーターが一気に満タンになる。抜けてから一定時間経つとまた同じ場所に生えてくる。
- 主人公は鼻の入り口より少し進んだところからスタートする。鼻の入り口より外に出てし待った瞬間にゲームオーバーが確定する。そのままゲームオーバーアニメーションに移行する。アニメーションは具体的には、鼻の入り口から伸びる鼻水になって外に飛び出し、そのままヤギが口を開けて鼻水ごと吸い込まれて口の中に入ってしまう。ここで、ゲームオーバーUIを出して、リトライやタイトルに戻ることができる。
- ヤギの鼻の穴のちょうど耳あたりにぬいぐるみが挟まっていて、ここがゴールになる。ここで鼻毛をつかむボタンを押すことでぬいぐるみを引っこ抜き始めるが、同時にヤギのくすぐりメーターも上昇してしまう。ぬいぐるみにも耐久メーターがあり、引っこ抜いている間はちょっとずつ耐久メーターが下がっていく。この耐久がゼロになることでゲームクリアが確定し、ここからクリアアニメーションがスタートする。クリアアニメーションは、まず鼻の中が主人公含めて見えなくなると同時にヤギが通常よりさらに少し大きなくしゃみをし始める(この時、耳のちょうど下の当たりに主人公がいて、そこがちょうど常に中心にくるようにカメラを動かしつづける。)。くしゃみをすると同時にぬいぐるみを抱いた主人公が大きなヤギの口から飛び出してくる。ヤギの口から出てきて一秒後ぐらいにゲームクリアとして、UIを表示する(ゲームオーバーと同じ選択しを用意する)。
